# 🏫 SMA Negeri 8 Medan Front-End Website

This project is a Front-End user interface development for the official website of **SMA Negeri 8 Medan**. Built from scratch without frameworks (Pure HTML, CSS, and JS) to ensure fast performance, clean code structure, and a modern design.

## ✨ Key Features

This project incorporates various modern UI/UX interactions, including:
- **Responsive Design:** The layout automatically adjusts for PC, Tablet, and Mobile screens.
- **Scroll Reveal Animations:** Page elements appear dynamically with smooth animations as you scroll.
- **Interactive Navigation:** A sticky navbar with transparency effects and organized dropdown menus.
- **Dynamic Gallery:** A gallery page featuring smart category filtering and a Lightbox effect (images enlarge upon clicking).
- **Pagination System:** A functional JavaScript-based pagination system for the teacher and staff data tables.
- **Interactive Modals:** Elegant pop-ups on the Extracurricular page displaying detailed information and social media links.
- **Accordion FAQ:** A toggle system for the Frequently Asked Questions and financial transparency sections.

## 🛠️ Tech Stack

- **HTML5:** Semantic web page structure.
- **CSS3:** Custom styling, CSS Grid, Flexbox, and Keyframe Animations.
- **Vanilla JavaScript:** DOM (Document Object Model) manipulation for all functional logic and interactions without external libraries.
- **Font Awesome:** Icon library (vector).
- **Google Fonts:** Using "Lexend" and "Baumans" typefaces.

The project consists of several main pages:

5. `pages/sarana-prasarana.html` - School Facilities
6. `pages/struktur-organisasi.html` - Organizational Structure
7. `pages/staff.html` - Faculty & Management List
8. `pages/berita.html` - Latest News Portal
9. `pages/agenda.html` - Event Calendar
10. `pages/galery.html` - Photo Documentation
11. `pages/keuangan-bos.html` - Financial Transparency (BOS Fund)
12. `pages/ekstrakurikuler.html` - Extracurricular List & Detail Modals
13. `pages/layanan.html` - Academic Classes and Services

## 🚀 How to Run

Since this project uses pure programming languages, you do not need to install any packages or build tools (like npm/Node.js).

1. Clone this repository: `git clone https://github.com/Ainpersonal/FRONT-END-SMA-NEGERI-8-MEDAN.git`
2. Open the project folder.
3. Open `index.html` using your preferred web browser (Chrome, Edge, Firefox, Safari).
4. *(Optional)* It is highly recommended to open it using the **Live Server** extension in VS Code to ensure directory paths load perfectly.
